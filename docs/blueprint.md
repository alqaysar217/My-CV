# **App Name**: CosmicVCard

## Core Features:

- Bilingual Content Management: Manage website content in both English and Arabic, ensuring RTL layout for Arabic.
- Hero Section Animation: Implement the animated 3D space background, orbit rings, floating spheres, and neon styling as described, closely matching the provided image.
- Download CV: Allows users to download Mahmoud's CV.
- Contact via WhatsApp: Direct link to contact Mahmoud via WhatsApp.
- Data Storage: Data such as projects, certificates and books should be stored in Firestore to be rendered on the site
- Contact Form Submission: Capture messages via the contact form and store them in Firestore.

## Style Guidelines:

- Primary color: Electric Blue (#00AFFF) to evoke a futuristic and tech-driven aesthetic.
- Background color: Very dark desaturated purple (#0A031A) to create a dark galactic theme.
- Accent color: Neon Cyan (#19D3FF) used sparingly to highlight key elements and calls to action, creating contrast and drawing attention.
- Body and headline font: 'Space Grotesk' sans-serif font for all text, giving a modern, digital feel.
- Line icons with neon glow effect, enhancing the futuristic theme, on hover.
- RTL (Right-to-Left) support for the Arabic version, ensuring proper text alignment and reading direction.
- Subtle hover animations and parallax effects to enhance user experience.
